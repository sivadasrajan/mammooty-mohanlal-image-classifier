## How to configure
**1.** Installing pip [Python Package Manager]

```shell
$ sudo apt-get install python3-pip
```

**2.** Clone this repository to your local drive

```shell
$ git clone 
```

**3.** Install dependencies

```shell
$ pip3 install -r requirements.txt
```

**4.** Run the Jupyter Notebook
